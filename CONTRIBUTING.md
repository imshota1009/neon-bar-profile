# このプロジェクトへの協力方法

`neon-bar-profile` へ興味を持っていただき、ありがとうございます！このプロジェクトは、みなさまからのご意見やお手伝いを歓迎しています。

## 参加の方法

### 1. 提案や問題の報告（Issue）
何か気づいた点や、新しく追加したい機能のアイデアがあれば、[掲示板（Issues）](https://github.com/imshota1009/neon-bar-profile/issues)から新しくトピックを作成して教えてください。

### 2. 変更の提案（Pull Request）
自分でプログラムを直してみたい場合は、以下の手順でお願いします。
1. この保管庫を自分の手元にコピー（フォーク）します。
2. 新しい作業用スペース（ブランチ）を作ります。
3. 変更を保存し、手元の変更履歴を記録（コミット）します。
4. インターネット上のあなたの保管庫に送り（プッシュ）、こちらに向けて「変更を取り込んでほしい」というリクエスト（Pull Request）を送ってください。

## 守ってほしいルール
* 誰に対しても丁寧で優しい言葉遣いを心がけてください。
* 変更を加える際は、コードが見やすく、整理されている状態を保つようにお願いします。
